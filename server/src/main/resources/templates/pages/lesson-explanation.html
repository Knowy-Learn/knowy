Nombre lección

En Java, un arreglo es una estructura de datos que permite almacenar múltiples elementos del mismo tipo en una sola variable. Los arreglos tienen tamaño fijo (no se pueden expandir una vez creados) y cada elemento se accede mediante un índice que empieza en 0. Se declaran y crean así:

int[] numeros = new int[5]; // arreglo de 5 enteros

o con valores definidos:

String[] nombres = {"Ana", "Luis", "Pedro"};
